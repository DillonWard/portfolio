### How to run
#### Prerequisites 
- [Docker](https://docs.docker.com/get-docker/)
- [Docker Compose](https://docs.docker.com/compose/install/)

```
git clone https://github.com/DillonWard/portfolio.git
cd infrastructure
docker compose build
docker compose up
```
